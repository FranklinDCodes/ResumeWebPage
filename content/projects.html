Quantum Sudoku Solution Searcher

Qiskit in Python

  • Aims to solve a few missing squares on a sudoku game by searching with quantum advantage
  • Used the Qiskit Python library to write a quantum algorithm
  • Employs Grover's search algorithm to search possible solutions
  • Runs in √N time, where N is all possible correct and incorrect combinations
  • Program was too large to simulate or run on any current quantum computer

Check out the notebook: quantum_sudoku.ipynb
